# Sales Analytics Dashboard (Power BI)

## Project Overview
This project analyzes sales data using Power BI to evaluate revenue performance, profitability, customer segments, and product category contribution.

The dashboard helps businesses track sales KPIs and identify trends across regions and product categories.

## Key Features
- Regional sales performance analysis
- Product category contribution
- Profit and revenue tracking
- Customer segment analysis
- Quarterly sales trends

## Tools Used
Power BI  
DAX  
Excel  
Data Visualization  

## Key Insights
- Identified top performing regions driving revenue
- Highlighted product categories contributing the most profit
- Analyzed seasonal sales patterns and quarterly trends
